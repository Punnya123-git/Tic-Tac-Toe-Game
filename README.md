# 🎮 Tic-Tac-Toe (XO Matrix)

A simple and interactive Tic-Tac-Toe web application built using **HTML, CSS, and JavaScript**.
This project focuses on implementing game logic, DOM manipulation, and enhancing user experience with visuals and sound effects.

🔗 **Live Demo:** https://xo-matrix.netlify.app/

---

## 🚀 Features

* ✅ Responsive UI (works on different screen sizes)
* 🎯 Winner detection with highlighted winning pattern
* 🤝 Draw detection
* 🔄 Reset & New Game functionality
* 🔊 Sound effects for clicks, win, and draw
* 🎉 Celebration animation on winning

## ⚙️ How It Works

* Players take turns placing **X** and **O**
* The app checks predefined **winning patterns**
* If a player wins:

  * Winning boxes are highlighted
  * Celebration animation + sound is triggered
* If all boxes are filled:

  * Game is declared a **draw**
* Users can reset or start a new game anytime
